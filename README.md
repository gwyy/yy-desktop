# 云烟工作台

## 背景
作为一名开发者，每天都要浏览大量的网站，浏览器收藏夹的确能够解决我很大一部分问题，但是当我的收藏夹有了几百个收藏的时候，每天经常被访问的收藏可能就有几十个、我需要一个 “工作台上面的快速响应区”， 而不是每次都从书架中找到对应的网站。用了市面上很多导航，定制化程度比较低，而且数据也无法保证不会丢失，于是，想要开发一个自己的导航站的想法开始了。
